jQuery SelectMoveOptions

Mova options entre selects HTML com este plugin jQuery com estilização Bootstrap.

Download (3.1kb)
Tweet

Descrição

O jQuery SelectMoveOptions é um plugin desenvolvido utilizando o jQuery com a estilização usando as classes disponíveis no Twitter Bootstrap, onde você pode mover options entre dois selects HTML utilizando o duplo click em cada options ou utilizando as setas de controle.

Instalação

Inclua o arquivo bootstrap-selectmoveoptions.js em seu código html.

<script src="js/bootstrap-selectmoveoptions.js"> </script>

Dependências

jQuery

Bootstrap

Opções

height string Altura em pixels dos selects (ex: '250px')
disableFirst boolean Desabilita o primeiro select. Nenhum option pode ser removido do mesmo
disableLast boolean Desabilita o primeiro select. Nenhum option pode ser removido do mesmo
